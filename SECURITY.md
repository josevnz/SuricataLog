# Security Policy

## Supported Versions

I only do bug fixes and security checks on the latest version of SuricataLog.

| Version  | Supported          |
|----------|--------------------|
| 0.0.9    | :white_check_mark: |
| <= 0.0.8 | :x:                |

## Reporting a Vulnerability

Please report an issue and provide as much detail as possible.
